# Nest Payments

A payment record system built with Nest.js and Stripe.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Api Documentation

To view the open api documentation use:

```bash
$ npm run start:dev
```

and navigate to `localhost:3000/api`.
